# Life Tracker

Автоматизированный трекер привычек, финансов и целей в Google Sheets.
Разворачивается одной командой — таблица создаётся, меню появляется, триггеры встают сами.

## Что создаётся

- 🏠 Главная — сводка за день/месяц
- ✅ Привычки — ежедневный трекер с процентами
- 💰 Финансы — доходы и расходы
- 🎯 Цели — прогресс по целям
- 📅 Планер — задачи на неделю
- 📜 История — лог всех изменений

## Запуск

```
pip install gspread google-auth google-api-python-client
python tracker_setup.py
```

Нужен `credentials.json` от Google Service Account. Инструкция — в `tracker_README.txt`.
